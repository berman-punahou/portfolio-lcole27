Terms to Know

  1. HTML - short for Hypertext Markup Language. A coding language used to design websites by displaying elements.
  2. CSS - short for Cascading Style Sheets. A coding language used in conjuction with HTML to style elements appearing on a web page.
  3. Bootstrap - a library connected to HTML, CSS, and JavaScript (another coding language). It can be used to add different pieces of code to web pages, such as a grid or photo carousel.
  • Elements - used in HTML to define different types of content. Usually thet have a start and end tag. Common types are:
  •   head (contains document metadata)
  •   body (everything appearing on page)
  •   div (separates different pieces of code)
  •   h1, h2, h3 (headings)
  •   paragraph
  •   ul, ol (lists)
  • Attributes - used in HTML/CSS to alter elements. Added to the opening tag of an element or included in the CSS stylesheet.
  • CSS Selectors - used to differentiate what kind of styling will be applied to different sections of code.
  •   Element - all elements of a certain type will have that styling
  •   Class (.) - all members added to a class will have a certain styling
  •   ID (#) - anything in the selector will have a unique styling
